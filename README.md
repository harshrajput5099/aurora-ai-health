# 🌟 Aurora – AI Health Companion

Aurora is an AI-powered cross-platform health and wellness companion designed to help users build healthier lifestyles through intelligent tracking, personalized insights, and voice-enabled health logging. Built using React Native and Expo, Aurora combines modern mobile development with Generative AI technologies to deliver a seamless and engaging health management experience across Android and iOS devices.

---

## 📱 Overview

Maintaining healthy habits consistently can be challenging. Aurora addresses this problem by providing a centralized platform where users can monitor key wellness metrics, receive AI-generated recommendations, and log health activities using natural voice interactions.

The application integrates advanced AI capabilities through Claude Haiku and OpenAI Whisper to offer personalized wellness guidance and frictionless health tracking.

## 🚀 Installation

### Prerequisites

* Node.js
* npm or yarn
* Expo CLI
* Firebase Project
* Claude API Key
* OpenAI API Key

### Clone Repository

```bash
git clone https://github.com/yourusername/aurora-ai-health.git
cd aurora-ai-health
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
FIREBASE_API_KEY=YOUR_FIREBASE_KEY
FIREBASE_AUTH_DOMAIN=YOUR_FIREBASE_DOMAIN
FIREBASE_PROJECT_ID=YOUR_PROJECT_ID

CLAUDE_API_KEY=YOUR_CLAUDE_API_KEY
OPENAI_API_KEY=YOUR_OPENAI_API_KEY
```

### Run Application

```bash
npx expo start
```
